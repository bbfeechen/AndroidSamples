SR2.0 同梱apk (BaseApp-130812ベース)

EYS:        Ver:32220_01.2013071201(RV)
ICH:        Ver:34134_00.2013080901
SmartPhone: Galaxy Nexus (Android4.2)


1. 機材セットアップ
   カメラとスマホを一度接続し、スマホにWi-Fi接続情報を記憶させる

   カメラ、サスペンドに入れる
   スマホ、タスクリストからPMMを殺しておく

   スマホ側をEclipse ADTやAndroid Monitorに接続して、ログをとれるようにする。
   過去のログはフラッシュしておくこと。

2. カメラ起動

   カメラ起動。
   アプリリストに入る。

   ICHの場合はエントロピーが不足するので、アプリリストでカーソルを右往左往10秒ほどしておく。

   SmartRemoteを立ち上げる。

3. PMM起動

   SmartRemoteのWi-Fiセットアップ画面で、クルクルが止まった瞬間にPMMを立ち上げる。

   厳密にはこのタイミングは測定には影響しないはずだが、Androidが裏で接続しに行ってしまうことがあり、
   PMMが接続エラーとすることがある。
   経験則的に、このタイミングが一番スムーズだと感じた。

4. ログとり
  PMM側にEE画が出力されていることを確認する。
  タグSR_T_Logでフィルターし、該当ログだけを保存する。

5. 終了

   SmartRemoteを終了し、カメラをサスペンド。
   PMMを終了し、タスクリストからPMMを殺す

   Android MonitorもしくはEclisep ADTのログをフラッシュしておく
